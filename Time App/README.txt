Flutter Time App:

	    Simple clock and stop watch mobile app that utilizes a Flutter analog clock package to display
     different times of major cities around the world all with custom and unique designs to the analog clocks.
    Through the navigation bar the user can navigate to the stopwatch screen that utilizes a Flutter stop watch
    timer package to create a stop watch that can start, stop, reset and lap on time.
